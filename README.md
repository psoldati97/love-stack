## Quick Install (5 minutes)

### 1. Clone this repo

```bash
git clone https://github.com/ondecksociety/love-stack.git
cd love-stack
```

### 2. Run the installer

```bash
chmod +x install.sh
./install.sh
```

The installer asks which track you want (Personal, Solopreneur, or Developer) and installs the appropriate skills, templates, and configurations.

### 3. Verify

```bash
chmod +x verify.sh
./verify.sh
```

This checks that all skills, MCPs, and configurations are properly installed.

## Persistent Memory

MemPalace is included in the Developer track. It stores every Claude exchange in a searchable local vault, organized as wings (projects/people), rooms (sub-topics), and halls (memory types). On session start, Claude loads ~170 tokens of critical facts, then searches the vault as needed. No API keys. Runs offline. 29 MCP tools. 96.6% raw recall on LongMemEval.

## The Story

On Deck Society is a Public Benefit Corporation in Phoenix, AZ. Our CEO had zero programming experience. Our CTO has a day job. Our CCO is a designer.

In January 2026, our CEO, Pacifico Soldati, sat down with Claude.ai and vibe coded the MVP of Love On Deck in 20 days. Authentication, real-time messaging, a patent pending filtering engine, payments, admin dashboard, and Row Level Security across every table. 10,000 lines of production code.

Then after Opus 4.6 was released, Pacifico began using Claude Code to upgrade the MVP and since then has collected a variety of helpful tools, skills, MCPs, CLIs, and more. After sharing the stack on Threads, Pacifico was encouraged to turn it into a plugin and the Love Stack was born. v3.2 adds MemPalace persistent memory, PostHog analytics, Playwright automation, and 15 new skills covering accessibility, responsive design, GDPR compliance, parallel debugging, brand voice, and more.

We could charge for this. Plenty of people sell Claude skill packs for $50-$200. We're giving it away because we're a Public Benefit Corporation and that's how we roll.
